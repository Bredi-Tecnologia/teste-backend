Este projeto é a resposta da prova baseado em laravel 8

## Instruções para Rodar

- git clone repositorio

- Editar .env apartir de .env.example

- composer install

- php artisan key:generate

- php artisan migrate

- php artisan db:seed

- php artisan serve
